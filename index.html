การเชื่อมต่อ Firebase (สำหรับเก็บประวัติย้อนหลัง)

เพื่อให้ทีมดูข้อมูลย้อนหลังได้จากหลายเครื่อง ต้องตั้งค่า Firebase Firestore:
1. เข้า Firebase Console → สร้างโปรเจกต์ใหม่ (หรือใช้โปรเจกต์เดียวกับระบบ deposit-form-app เดิม)
2. เปิดใช้ Firestore Database (โหมด production)
3. ไปที่ Project Settings → Web App → คัดลอกค่า config
4. แก้ไขไฟล์ firebase-config.js ในโปรเจกต์นี้ ใส่ค่าที่คัดลอกมา แล้ว deploy ใหม่
